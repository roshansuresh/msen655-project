Winkler Method for Fusion of Information Sources¶

  • This notebook presents the Winkler method for the fusion of multiple information sources, represented as Gaussian Processes.
  • The method follows the procedure in Winkler RL (1981) Combining probability distributions from depen- dent information sources. Manag Sci 27(4):479–488
  • Two Gaussian Processes will be created and fused using the Winkler Method. (Just like in the actual optimization problem)
  • The design space and objective space are considered to be one dimensional.

### Define Projection Function 
# This function takes the samples out of the LHS function and transforms them into uniform samples in the actual design space
def sample_project(x, x_range):
    n_samp = len(x)
    
    x_lower = x_range[0]
    x_upper = x_range[1]
    
    x_projected = np.zeros(n_samp)
    for i in range(n_samp):
        x_projected[i] = x[i]*(x_upper - x_lower) + x_lower
    
    return x_projected

Create and train Gaussian Processes¶

  • Now that the training samples for the two GPs have been generated. the two GPs are trained
  • The first GP is trained to simulate function f_1(x) = -(1.6 - 3x)sin(18*x)
  • The second GP is trained to simulated function f_2(x) = -(1.8 - 3x)sin(18*x + 0.1)
  • The Matern Kernel is used for both GPs
  • The hyperparameters are estimated using Maximization of the Log Marginal Likelihood using the L-BFGS algorithm

Perform Winkler Fusion¶

  • Once the GPs are trained, Winkler Fusion is performed
  • Winkler Fusion uses a different set of design samples. This design sample set is also generated using Latin Hypercube Sampling
  • The Winkler Means for the samples are taken as the reference response values to train the fused GP

# Define Number of samples for fusion
n_samples_fusion = 60

# Generate Fusion Samples using Latin Hypercube Sampling
x_fuse_lhs = lhs(1, samples=n_samples_fusion, criterion='centermaximin') 

# Project the samples into the actual design space
x_fuse_proj = sample_project(x_fuse_lhs, [x_low, x_up])

### Define the function that computes the Winkler means
def Winkler_fusion(x, gp_1, gp_2, sigma_f1, sigma_f2):
    n_des = len(x)
    
    gp1_means = np.zeros(n_des)
    gp1_stds = np.zeros(n_des)
    gp2_means = np.zeros(n_des)
    gp2_stds = np.zeros(n_des)
    gp_means = np.zeros([n_des,2])
    gp_vars = np.zeros([n_des,2])
    
    [gp1_means, gp1_stds] = gp_1.predict(x.reshape(-1,1), return_std=True)
    [gp2_means, gp2_stds] = gp_2.predict(x.reshape(-1,1), return_std=True)
    gp_means = np.column_stack((gp1_means, gp2_means))
    
    var_1 = gp1_stds**2 + sigma_f1
    var_2 = gp2_stds**2 + sigma_f2
    gp_vars = np.column_stack((var_1, var_2))
    
    rho_tilde = np.zeros([n_des,2,2])
    rho = np.zeros([n_des,2,2])
    covar = np.zeros([n_des,2,2])
    mean_winkler = np.zeros(n_des)
    var_winkler = np.zeros(n_des)
    
    for i in range(n_des):
        # Compute rho_tilde matrix for current design sample
        rho_tilde_x = np.zeros([2,2])
        mean_vec = gp_means[i,:]
        var_vec = gp_vars[i,:]
        for j1 in range(2):
            for k1 in range(2):
                mean_j = mean_vec[j1]
                mean_k = mean_vec[k1]
                var_j = var_vec[j1]
                var_k = var_vec[k1]
                rho_tilde_x[j1,k1] = np.sqrt(var_j)/(np.sqrt((mean_j - mean_k)**2 + var_j))
    
        # Compute correlation matrix for current design sample
        rho_x = np.zeros([2,2])
        #var_vec = gp_vars[i,:]
        for j2 in range(2):
            for k2 in range(2):
                var_j = var_vec[j2]
                var_k = var_vec[k2]
                rho_x[j2,k2] = (var_k/(var_j + var_k))*rho_tilde_x[j2,k2] + (var_j/(var_j + var_k))*rho_tilde_x[k2,j2]
        
        # Compute covariance matrix for current design sample
        covar_x = np.zeros([2,2])                                             
        for j3 in range(2):
            for k3 in range(2):
                std_j = np.sqrt(var_vec[j3])
                std_k = np.sqrt(var_vec[k3])
                covar_x[j3,k3] = rho_x[j3,k3]*std_j*std_k
        
        # Compute Winkler Mean
        e_vec = np.ones(2)
        mean_winkler[i] = (np.linalg.multi_dot([np.transpose(e_vec), np.linalg.inv(covar_x), mean_vec]))/(np.linalg.multi_dot([np.transpose(e_vec), np.linalg.inv(covar_x), e_vec]))
        var_winkler[i] = 1/(np.linalg.multi_dot([np.transpose(e_vec), np.linalg.inv(covar_x), e_vec]))
        rho_tilde[i,:,:] = rho_tilde_x
        rho[i,:,:] = rho_x
        covar[i,:,:] = covar_x
    
    return mean_winkler, var_winkler

## Compute the Winkler means for the generated design samples
[y_wink, var_wink] = Winkler_fusion(x_fuse_proj, GP1_post, GP2_post, var_fid1, var_fid2)
